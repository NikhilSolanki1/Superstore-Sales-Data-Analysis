# Superstore-Sales-Data-Analysis
## Project Objective
To Contribute to the success of Business by utilising data analysis techniques, specifically focusing on time series analysis, to provide valuable insights and accurate sales forecasting

## Dataset used
- <a href="https://github.com/NikhilSolanki1/Superstore-Sales-Data-Analysis/blob/main/SuperStore_Sales_Dataset.csv">Super Store Dataset</a>

## Questions (KPIs)
1. Which Payment mode is used in different region?
2. Which region had the most sales?
3. In which year did the company had more sales?
4. What Products were mostly bought by customers?
5. What is the total Profit?
6. Forecast the sales for future 15 days
7. Which country will have most sale based on forecast?
